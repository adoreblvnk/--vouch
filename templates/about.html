--vouch Features

Simple & Efficient

--vouch features simple yet efficient data processing.

0 Document Storage

Despite being seemingly difficult, vouching requires a small amount of data to work. Hence, --vouch stores only a portion of extracted data. Hence, it's privacy focused as well.

Open Source

--vouch's API is open source, meaning you can use --vouch on your home machine for free. --vouch's API is on Github, check the footer for the link.

Secure

All data stored are on a necessity to use basis. All data pertaining to documents with the "encrypted" tag are encrypted. Data retention of 6 months.
